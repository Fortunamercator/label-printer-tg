# Telegram Mini App — Печать этикеток

Мини-приложение для Telegram, которое позволяет быстро формировать и печатать этикетки для товаров.

## Возможности

- Выбор товара из списка шаблонов
- Указание количества этикеток
- Автоматическая подстановка текущей даты (с возможностью изменения)
- Генерация PDF с этикетками
- Превью и печать одной кнопкой
- Сохранение последних введённых значений

## Установка

### 1. Загрузка на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в репозиторий
3. Перейдите в **Settings** → **Pages**
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`
5. Нажмите **Save**
6. Дождитесь деплоя (обычно 1-2 минуты)
7. Ваш URL будет: `https://ваш-username.github.io/название-репозитория/`

### 2. Создание Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен (он понадобится позже)

### 3. Подключение Mini App

1. В @BotFather отправьте `/newapp`
2. Выберите вашего бота
3. Введите название приложения (например: "Печать этикеток")
4. Введите описание
5. Загрузите иконку (640x640 px)
6. Введите URL вашего GitHub Pages сайта
7. При необходимости укажите короткое имя для прямой ссылки

### 4. Добавление кнопки меню

1. В @BotFather отправьте `/setmenubutton`
2. Выберите вашего бота
3. Введите URL вашего Mini App
4. Введите текст кнопки (например: "Открыть")

## Редактирование шаблонов

Шаблоны этикеток хранятся в файле `data/templates.json`.

### Формат шаблона

```json
{
  "id": "197",
  "name": "Пасхальная подставка для яиц под кулич",
  "size": "30x30см",
  "color": "Натуральный",
  "country": "Россия",
  "manufacturer": "ИП Кванталиани Н.А.\nТульская область, рп. Заокский",
  "material": "Маньчжурский ясень",
  "barcode": "2043750568169"
}
```

### Поля шаблона

| Поле | Описание |
|------|----------|
| `id` | Номер полки/артикул (отображается в списке выбора) |
| `name` | Название товара |
| `size` | Размер (выводится крупно справа) |
| `color` | Цвет |
| `country` | Страна производства |
| `manufacturer` | Производитель (можно использовать `\n` для переноса строки) |
| `material` | Материал |
| `barcode` | Штрих-код EAN-13 (13 цифр) |

### Добавление нового шаблона

1. Откройте `data/templates.json` на GitHub
2. Нажмите кнопку редактирования (карандаш)
3. Добавьте новый объект в массив `templates`
4. Сохраните изменения (Commit changes)

## Настройка размера этикетки

Размер этикетки задаётся в файле `js/pdf-generator.js`:

```javascript
const labelWidth = 58;  // ширина в мм
const labelHeight = 40; // высота в мм
```

Измените эти значения под ваши этикетки.
